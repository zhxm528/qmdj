#排地八神
##排盘规则
- “地八神”是指：值符、腾蛇、太阴、六合、白虎、玄武、九地、九天
- 排“地八神”指的是在奇门遁甲起局时，把八个“神”（值符、腾蛇、太阴、六合、白虎、玄武、九地、九天）依照阴遁/阳遁的行走顺序，依次分布到九宫（去中宫）中的过程。
##排布要点（操作步骤）
- 先找到时柱对应的旬首，并在后台打印"旬首”。
- 按照时柱定旬首对应的地盘干，该地盘干对应的宫为地八神的起点：值符，并在后台打印”值符“。
- 以值符所在的之宫为起点，在九宫上按八神固定顺序排布：值符 → 腾蛇 → 太阴 → 六合 → 白虎 → 玄武 → 九地 → 九天。
- 阳遁顺行、阴遁逆行，逐宫落到地盘，不入中宫：只排外圈八宫。

##核心功能
- 创建后台程序 app/api/dibashen/route.ts 
- 前台页面： app/page.tsx  
- 前台页面中，点击"排盘"后，在排完天盘干后，执行排地八神
- 排盘顺序：定局 → 地盘干 → 天盘干 → 地八神
