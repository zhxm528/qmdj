# 获取“地支藏干表”
从已知的四柱八字，获取“地支藏干表”：**先确定四柱的四个地支（年支、月支、日支、时支）→ 再按固定映射表把每个地支对应的藏干列出来**。
这张映射表本身是传统命理里的常用基础表（基本不随人而变），你只要“查表”即可。



## 1）最常用的地支藏干表（按“主气→中气→余气”顺序）

| 地支 | 藏干    |
| -- | -- |
| 子  | 癸     |
| 丑  | 己、癸、辛 |
| 寅  | 甲、丙、戊 |
| 卯  | 乙     |
| 辰  | 戊、乙、癸 |
| 巳  | 丙、戊、庚 |
| 午  | 丁、己   |
| 未  | 己、丁、乙 |
| 申  | 庚、壬、戊 |
| 酉  | 辛     |
| 戌  | 戊、辛、丁 |
| 亥  | 壬、甲   |

> 说明：这里的排列顺序通常代表“主藏（主气）”在前，其次为中气、余气。不同流派在“强弱权重”上会有细分，但“藏哪些干”这件事基本一致。



## 2）你拿到四柱后怎么用（实际操作步骤）

假设你四柱是：**年柱 X子、月柱 X寅、日柱 X午、时柱 X辰**（只看地支）

1. 取四个地支：子、寅、午、辰
2. 查表：

   * 子 → 藏 癸
   * 寅 → 藏 甲、丙、戊
   * 午 → 藏 丁、己
   * 辰 → 藏 戊、乙、癸
3. 把这些藏干按柱归类写出来（年支藏干/…），后续才能做：

   * 判“根”（某天干在地支里有没有根）
   * 判“透”（藏干有没有出现在天干上）
   * 排十神（相对日主）
   * 看合冲刑害时，判断“被合走/被冲开”牵动哪些藏干
