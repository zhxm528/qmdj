实现的功能
1. 状态管理
添加了 ganMetaData state 用于存储天干五行阴阳表数据
2. API 调用
在 handleBaziPaipan 函数中，获取四柱信息后调用 /api/bazi/wuxingyinyang API
提取四柱的天干（年干、月干、日干、时干）
使用 POST 请求获取对应的天干五行阴阳信息
包含错误处理，失败时仅警告，不影响主流程
3. 表格展示
在步骤1的展示区域添加了"天干五行阴阳表"表格
表格包含4列：柱位、天干、阴阳、五行
显示四柱的天干及其对应的阴阳和五行属性
表格样式与地支藏干表一致
表格位置在天干五行阴阳表之后、地支藏干表之前
4. 数据重置
在开始新的排盘时重置 ganMetaData 为 null
现在在步骤1（定日主）的结果中会显示：
日主信息和五行分布
五行统计雷达图
天干五行阴阳表格（新增）
地支藏干表格
