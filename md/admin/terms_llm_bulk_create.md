# 术语批量创建（LLM）

## 目标
在后台术语管理中通过 LLM 批量生成术语并写入数据库。

## 页面入口与交互
- 入口位置：管理页面的术语列表页，新增按钮“批量增加”。
- 点击按钮后弹出弹框，包含：
  - 术语分类选择器（单选）
  - 文本输入框（多行）
  - 提交按钮
  - 取消按钮

## 提交流程
- 点击“提交”后，调用后台独立程序。
- 将文本框内容与所选分类作为请求参数传给后台。
- 后台调用 LLM 接口，返回多个“关键字/术语候选”。

## 数据落库规则
- 对 LLM 返回的每个关键字，按术语表字段写入数据库。
- 分类字段使用用户在弹框中选择的分类。
- 其他字段按默认策略填充（由后台定义）。

## 失败处理
- LLM 请求失败时，返回错误提示，不新增数据。
- 数据库写入失败时，记录错误日志并提示用户。
