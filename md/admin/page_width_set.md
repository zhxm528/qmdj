## 用户管理页面宽度调整规则

### 说明
已修改用户管理页面容器，使其自适应撑满主内容区。

### 修改内容
- 文件：`app/admin/**/page.tsx`（app/admin 下所有子目录页面）
- 修改前：`<div className="max-w-7xl mx-auto">`（最大宽度 1280px，居中）
- 修改后：`<div className="w-full">`（宽度 100%，撑满主内容区）

### 效果
- 页面容器会撑满主内容区的可用宽度
- 不再受 `max-w-7xl`（1280px）限制
- 表格有更多空间显示内容，减少水平滚动

### 验证
- 代码已通过 linter 检查，无错误
- 布局从固定最大宽度改为自适应宽度
- 现在用户管理页面会自适应撑满主内容区的宽度，充分利用可用空间
