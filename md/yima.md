#排驿马
## 排盘规则
### 确定时柱的地支，即"时支"
### 根据下面口诀确定驿马所对应的地支
- 申子辰： 马 → 寅
- 寅午戌： 马 → 申
- 巳酉丑： 马 → 亥
- 亥卯未： 马 → 巳
### 地支对应的宫位
| 辰巳 |  午  | 未申 |
| 卯   |      |  酉 |
| 寅丑 |  子  | 亥戌 |
### 以地支对应的宫位把驿马排入相应的宫

## 核心功能
- 创建后台程序 app/api/yima/route.ts 
- 前台页面： app/page.tsx  
- 前台页面中，点击"排盘"后，在排完空亡后，执行排驿马
- 排盘顺序：定局 → 地盘干 → 天盘干 → 地八神 → 天八神 → 九星 → 八门 → 空亡 → 驿马
