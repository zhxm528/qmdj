#排八门
- “排八门”就是在起局时，把八个“门”（休、生、伤、杜、景、死、惊、开）按照阴遁/阳遁的行度，依次分布到八个宫位（不入中宫）上
##八门顺序
- 休 → 生 → 伤 → 杜 → 景 → 死 → 惊 → 开

##值使门
- 地八神值符所在宫对应的八门中的哪一门就是值使门。
##排法要点
- 从时干对应的旬首按照六十甲子顺序数到时辰干支，值使门以同样的计算步数，按照八卦（阳遁顺数: 坎1 → 坤2 → 震3 → 巽4 → 中5 → 乾6 → 兑7 → 艮8 → 离9, 阴遁逆数: 离9 → 艮8 → 兑7 → 乾6 → 中5 → 巽4 → 震3 → 坤2 → 坎1）顺序排入宫位。
- 在计算步数时不要跳过中宫，如果最终步数落到中宫，则把值使门寄入坤2宫；
- 以“值使门”所在宫为起点，按顺时针依次排入八门。
- 不入中宫：八门只布八个外围宫位，中宫不置门。
