###排地盘干
- "地盘干"就是指——固定在九宫格上的十天干（甲、乙、丙、丁、戊、己、庚、辛、壬、癸）
- 地盘干排盘的顺序为：戊、己、庚、辛、壬、癸、丁、丙、乙
- 地盘干的起始格子为：定居时，确定的是几局，就从第几个格子开始。
- 地盘干把"戊、己、庚、辛、壬、癸、丁、丙、乙"按照格子顺时针或逆时针排入九宫。
- 定居时，如果是阳遁则顺时针排入格子（坎1、坤2、震3、巽4、中5、乾6、兑7、艮8、离8），如果是阴遁则逆时针排入格子（离9、艮8、兑7、乾6、中5、巽4、震3、坤2、坎1）。
- 格子的数字在九宫的位置参考奇门遁甲的九宫。

###排地八神
####排“地八神”指的是在奇门遁甲起局时，把八个“神”（值符、腾蛇、太阴、六合、白虎、玄武、九地、九天）依照阴遁/阳遁的行走顺序，依次分布到九宫（去中宫）中的过程。
####排布要点（操作步骤）
- 起局后先定阴/阳遁与几局，排地盘干。
- 按照时柱定旬首对应的地盘干，该地盘干对应的宫为八神的起点：值符
- 以值符所在的之宫为起点，在地盘上按八神固定顺序排布：值符 → 腾蛇 → 太阴 → 六合 → 白虎 → 玄武 → 九地 → 九天。
- 阳遁顺行、阴遁逆行，逐宫落到地盘，不入中宫：只排外圈八宫。

###排天八神
####排“天八神”指的是在奇门遁甲起局时，把八个“神”（值符、腾蛇、太阴、六合、白虎、玄武、九地、九天）依照阴遁/阳遁的行走顺序，依次分布到九宫（去中宫）中的过程。
####排布要点（操作步骤）
- 起局后先定阴/阳遁与几局，排地盘干。
- 按照时柱定旬首对应的天盘干，该天盘干对应的宫为八神的起点：值符
- 以值符所在的之宫为起点，在地盘上按八神固定顺序排布：值符 → 腾蛇 → 太阴 → 六合 → 白虎 → 玄武 → 九地 → 九天。
- 阳遁顺行、阴遁逆行，逐宫落到地盘，不入中宫：只排外圈八宫。


###排九星
####“排九星”就是把奇门遁甲的九颗星（天蓬、天任、天冲、天辅、天英、天芮、天柱、天心、天禽）按阴遁/阳遁的步法，依次布到八宫上（只排外圈八宫），作为天盘星层的格局基调。
####九星顺序和含义
- 1、天蓬：水、谋略、勇悍、远行；阴重则盗险酒色。
- 2、天任：土、任重、稳健、可托付；惰性、拖延亦随之。
- 3、天冲：木、冲劲、突破、启动；躁急、冲突。
- 4、天辅：木（文）、辅佐、教化、策划、文书。
- 5、天英：火、光明、才华、名声、表现；虚荣、外炫。
- 6、天芮：土（湿）、病符、麻烦、瑕疵；也主医药护理。
- 7、天柱：金（硬）、隔阂、口舌、阻隔；亦可为“立柱撑持”。
- 8、天心：金（灵）、机巧、思维、精密、医疗刀具。
- 9、天禽：中和之土，多派系取其居中宫，总摄调和，不随行飞布。

####排布要点（操作步骤）
- 起局后先定阴/阳遁与几局，排地盘干和排天盘干。
- 八宫按照顺时针对应九星的关系：坎宫-天蓬、艮宫-天仁、震宫-天冲、巽宫-天辅、离宫-天英、坤宫-天芮、兑宫-天柱、乾宫-天心，天禽和天芮同宫，中宫不入星；
- 地八神的值符所在宫对应的九星，放到天八神的值符所在宫，作为九星的起点；
- 从起点，从天蓬星开始排，然后按照顺时针的方向依次排入九星（天蓬、天任、天冲、天辅、天英、天芮、天柱、天心，天禽星和天芮星同宫），不入中宫：只排外圈八宫。


###排八门
####“排八门”就是在起局时，把八个“门”（休、生、伤、杜、景、死、惊、开）按照阴遁/阳遁的行度，依次分布到八个宫位（不入中宫）上
####八门顺序和含义
- 1、休门：休养、调和、人事圆融，利出行旅行、求医调理、水事、招聘。
- 2、生门：生发、财气、增长，利求财、经营、农业医药、开拓与增产。
- 3、伤门：损伤、口舌、争执、诉讼，也可用于军警执法、动刀开刃等“以刚制事”。
- 4、杜门：闭塞、封存、保密、仓储，利隐藏、封堵、档案与风控，但不利公开推进。
- 5、景门：光明、名声、传播、火电，利宣传营销、考试文名、影视演出、线上曝光。
- 6、死门：停滞、终结、耗损，主病殇障碍；亦可用于“收尾、报废、殡葬、断舍离”。
- 7、惊门：惊扰、变动、情报安防，利突发应对、预警布防、抓取消息与数据。
- 8、开门：开启、通达、贵人、手续签核，利开张签约、升迁会晤、远行出入境。
####排法要点
- 定阴阳遁与几局：先按节气和时间，参考定局，定为阳遁/阴遁与“某几局”。
- 安地盘：先安地盘（地盘八宫与地盘天干）。
- 定“值使门”：以时干所落之宫中的那一门为“值使门”，即从旬首按照六十甲子，按照后天八卦（坎1 → 坤2 → 震3 → 巽4 → 中5 → 乾6 → 兑7 → 艮8 → 离9）顺序，从地盘的值符数到时柱所在的宫，该宫对应的门就是“值使门”；
- 顺/逆布门：阳遁顺排；阴遁逆排。常用门序是：休 → 生 → 伤 → 杜 → 景 → 死 → 惊 → 开（阴遁逆行之）。
- 以“值使门”所在宫为起点，按上序在各宫依次排入八门。
- 不入中宫：八门只布八个外围宫位，中宫不置门。


###排空亡
####以“甲×旬”为旬首，顺排10个地支，余下两支即为空亡：
- 甲子旬：配 子丑寅卯辰巳午未申酉 → 空亡：戌、亥
- 甲戌旬：配 戌亥子丑寅卯辰巳午未 → 空亡：申、酉
- 甲申旬：配 申酉戌亥子丑寅卯辰巳 → 空亡：午、未
- 甲午旬：配 午未申酉戌亥子丑寅卯 → 空亡：辰、巳
- 甲辰旬：配 辰巳午未申酉戌亥子丑 → 空亡：寅、卯
- 甲寅旬：配 寅卯辰巳午未申酉戌亥 → 空亡：子、丑
####简用法：先判你这一天属于哪个“甲×旬”，再按表查到当日的两支空亡。

###排驿马
####以年支或日支为主（常用日支），看其所属三合局，对应一枚固定的“马位”：
- 申子辰（水局）→ 驿马在 寅
- 寅午戌（火局）→ 驿马在 申
- 亥卯未（木局）→ 驿马在 巳
- 巳酉丑（金局）→ 驿马在 亥

###排寄宫
####寄宫定义
奇门遁甲九宫有中宫（五宫）而八门、八神只有八个，要“占八方不留中”，于是把“中宫的内容”按某条规则并入到某个边宫，这个动作就叫“寄宫”。
####排布要点
- 直接把中宫寄到坤二宫。
- 把中宫的天干寄到坤二宫。

###天干
- 天干就是十个“干支记时”的天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸。在奇门遁甲里，天干不仅是记时的符号，更是占断里非常重要的“用神/信号位”，它们会落到九宫里，与八门、九星、八神发生生克制化，从而解读人事吉凶。
- 天干的奇门常用象意
甲（阳木）：主领袖、本体、事之根本、开始；在局中常当“用事核心/问事者”。
乙（阴木，三奇）：柔韧、关系、人脉、策划、技术巧思、植物藤蔓；多主通达与机巧。
丙（阳火，三奇）：光明、名气、曝光、行动力、军警武备、速度；多主显化与突破。
丁（阴火，三奇）：文书、票据、凭证、契约、小火、精致手艺；多主手续、文案、口碑。
戊（阳土，六仪）：屏障、权柄、规则、体系、房地产、稳定；可为保护/阻隔之象。
己（阴土，六仪）：湿土、筹划、细务、琐事、隐忧、牵绊；多主内耗或细密运营。
庚（阳金，六仪）：刚硬、刀兵、执法、决断、竞争、拆解；多主强力与冲突。
辛（阴金，六仪）：贵气、金融、饰物、精加工、审美、合规；多主精致与标准。
壬（阳水，六仪）：江海、大流量、运输、远行、平台/流量、变动；多主扩散与流通。
癸（阴水，六仪）：雨露、信息、秘密、润下、情绪、流言；多主隐秘与渗透。

###地支
- 地支就是传统干支纪时里的“十二支”：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。它们最初是记时与定方位的序号，后来扩展到历法、占筮、风水等体系中。
- 对应生肖：子鼠、丑牛、寅虎、卯兔、辰龙、巳蛇、午马、未羊、申猴、酉鸡、戌狗、亥猪。
- 对应时辰（约两小时一支）：
子(23–01) 丑(01–03) 寅(03–05) 卯(05–07) 辰(07–09) 巳(09–11)
午(11–13) 未(13–15) 申(15–17) 酉(17–19) 戌(19–21) 亥(21–23)
对应方位（以正北为子，顺时针）：子北、卯东、午南、酉西；寅艮、卯震、辰巽、巳巽偏南、未坤、申坤偏西、戌乾、亥乾偏北（不同体系细分略异，核心是子北午南卯东酉西）。
- 四季节气（月建）：
春：寅(正月) 卯(二月) 辰(三月)
夏：巳(四月) 午(五月) 未(六月)
秋：申(七月) 酉(八月) 戌(九月)
冬：亥(十月) 子(十一月) 丑(十二月)
- 五行与阴阳（常见归属）：
子(水, 阳) 丑(土, 阴) 寅(木, 阳) 卯(木, 阴) 辰(土, 阳) 巳(火, 阴)
午(火, 阳) 未(土, 阴) 申(金, 阳) 酉(金, 阴) 戌(土, 阳) 亥(水, 阴)

###八神
- 值符（总领、权柄、核心）主权威、统筹、门面与“官方说明”。落宫旺且得生，多主名正言顺、效率高。
- 螣蛇（幻象、反复、口舌）多虚惊、纠缠、反复无常、文字/代码里小坑、小瑕疵；谈判易绕弯子。
- 太阴（贵人、文书、内助）主细致、审稿、账务、背后助力、女性贵人、夜间/室内之利；适合文案、审核、精修。
- 六合（和合、交易、社交）主合作、签约、人脉撮合、媒介平台；谈合作、求调和、跑渠道时加分。
- 白虎（刀兵、冲突、损伤）主急躁、官非、血光、硬碰硬；做手术、维权、执法动作时强但带代价。
- 玄武（隐秘、盗取、私情）主秘密、信息截取、暗箱、物流/快递、账号安全；宜查漏、做渗透测试，不宜大张旗鼓。
- 九地（稳固、掩护、土地）主稳定、隐藏、防守、地产/基础设施；适合打地基、封网收口、风控内控。
- 九天（高远、速度、突破）主上升、扩散、航天航空、远程与“提权”；适合宣传放大、远行、提速突破。

###八门
八门是奇门遁甲盘中的“门盘”，描述事情发展的路径与结果倾向（好办/难办、公开/隐藏、快/慢等）。八个门分别是：
- 开门（金，吉）——开拓、公开、门面、审批通过、成交、启动发布。
- 休门（水，吉）——休养、缓和、人际融通、旅游、调解、放松复盘。
- 生门（土，吉）——增长、收益、客户、资源、健康、招财办事。
- 景门（火，吉）——曝光、名声、流量、视觉内容、策展、宣传上镜。
- 伤门（金，凶）——冲突、受伤、损耗、维权对抗、手术刀兵。
- 杜门（木，凶）——闭塞、封存、阻滞、卡点、审查、备案。
- 惊门（金，凶）——惊扰、突发、预警、跳变、舆情、惊吓与惊喜并存。
- 死门（土，凶）——停摆、报废、亏耗、旧系统、冷门、终止与收尾。

###九星
九星是奇门遁甲中的“星盘”，刻画事件的能力、技术、力度与成效方式。常用名与常见象意如下（括号内为五行）：
- 天蓬（水）——险悍、流动、远行、航运/流量、黑客/灰产、投机与偏财、酒色是非；旺则胆大能闯，衰则贪乱易漏。
- 天任（土）——承担、稳健、资源与人手、运营执行、耐力；主可持续与扛事。
- 天冲（木）——冲劲、速度、运动、突击、突破瓶颈；快、猛，但易冒进。
- 天辅（木）——辅佐、学习、咨询、文档、导师与方法论；策划与规范化强。
- 天英（火）——光彩、创意、颜值、曝光、舞台与品牌；主“出圈”与表现力。
- 天芮（土）——病弱、瑕疵、纠结、医药护理、售后与修补；宜查错补洞。
- 天柱（金）——对立、阻滞、法务文书、门槛指标、审核卡点；能立规矩也能卡人。
- 天心（金）——谋略、精算、武器/工具、技术中枢、谈判斡旋；主“精准与要害”。
- 天禽（土，中五）——居中统合、总控枢纽、调和各方；多位于中宫，主“定盘与折中”。

###空亡
空亡（又称“旬空”）是干支法里的一个判断规则：
在每个十日一组的“旬”里（甲乙丙丁戊己庚辛壬癸配十个地支），总会剩下两个地支没被配上，这两个“缺席的地支”就叫空亡。凡是用神（如某个天干、门、星、神、宫位）落在这两个地支上，称为落空，其象多为虚、迟、落空、名不副实、事不着落；但用于“隐匿、低调、收尾、清退、去旧”之事时，反而有可取之处。

###驿马
驿马（也叫“马星”）是古代命理里专指奔波、移动、变化、出行的一颗神煞。命局或流年触发驿马，多主出差旅行、搬家调动、换岗跳槽、留学出国、物流交通、跑业务等“动”的事情。

###寄宫
寄宫在奇门遁甲里，字面意思是“把某个要素寄住在某一宫位”。
因为盘中有些要素没有固定的宫位，它们会“随谁走到哪儿，就寄在哪一宫”，据此来判断事件作用的落点与强弱。

###24节气
####24节气是把太阳在黄道上一圈（360°）的运行按每 15° 划分为24个时间点的传统历法系统。每到一个点就是一个“节气”，用来标示季节更替、气候特征与农时。平均间隔约15–16天。
构成：24个名目中，交替分为 “节” 与 “中气”（含春分、夏至、秋分、冬至等）。在农历中，“中气”还用来判闰月。
####名称与顺序（按季节）
- 春：立春、雨水、惊蛰、春分、清明、谷雨（约2/3–5至4/19–21）
- 夏：立夏、小满、芒种、夏至、小暑、大暑（约5/5–7至7/22–24）
- 秋：立秋、处暑、白露、秋分、寒露、霜降（约8/7–9至10/23–24）
- 冬：立冬、小雪、大雪、冬至、小寒、大寒（约11/7–8至1/19–21）

####各节气主旨
- 立春/立夏/立秋/立冬：四季之始。
- 春分/秋分：昼夜等长；夏至昼最长、冬至昼最短。
- 雨水、谷雨：降水增、雨生百谷。
- 小满、芒种：夏熟作物灌浆、带芒作物可播。
- 小暑/大暑：一年中最热段。
- 白露/寒露/霜降：由凉转寒、露霜出现。
- 小雪/大雪：降雪渐多。
- 小寒/大寒：一年最冷段。

###八卦
####八卦是用三爻（— 为阳，-- 为阴）组成的八种基础卦象，是易学体系的“八个原子符号”。它们描画自然与人事的基本范畴，彼此叠加还能构成六十四卦。
####八卦一览
- 乾 ☰（三阳）——天，健，西北，父，金
- 坤 ☷（三阴）——地，顺，西南，母，土
- 震 ☳（阳在下）——雷，动，正东，长男，木
- 巽 ☴（阴在下）——风/入，渗透，东南，长女，木
- 坎 ☵（阳在中）——水，险，正北，中男，水
- 离 ☲（阴在中）——火，丽，正南，中女，火
- 艮 ☶（阳在上）——山，止，东北，少男，土
- 兑 ☱（阴在上）——泽/悦，口舌，正西，少女，金
####后天八卦
坎1 → 坤2 → 震3 → 巽4 → （中5） → 乾6 → 兑7 → 艮8 → 离9

###四柱
####四柱就是把一个人的出生时刻用干支历表示成四个柱子：年柱、月柱、日柱、时柱。每柱由一个天干 + 一个地支组成，共 8 个字，俗称“八字”。
####四柱各指什么
- 年柱（年干年支）：大环境、祖上根基、幼年运。
- 月柱（月干月支）：成长环境、父母与事业起点，权重很高（取月以节气为界）。
- 日柱（日干日支）：以日干为“日主/命主”（你自己），日支为“配偶宫”。
- 时柱（时干时支）：子女、晚景、潜能与落点（时支按两个小时一时辰换算）。


###符头
符头（也写作“值符头”）是奇门遁甲里的一个定位术语，指值符所落之宫的天盘天干。通俗点说：起好局后，先找值符落在哪一宫，再看该宫“上面那排”的天盘干，这个天干就是符头。

###六十甲子
六十甲子就是把十天干（甲乙丙丁戊己庚辛壬癸）与十二地支（子丑寅卯辰巳午未申酉戌亥）按固定顺序两两相配，形成的60个组合的循环，又叫干支纪法/六十花甲。因为10与12的最小公倍数是60，所以一轮为60。
