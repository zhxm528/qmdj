#排天盘干
##排盘规则
- "天盘干"是排在地盘干之上的另一层十天干
- 十天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
- 指定的外围八宫顺序：
 - 阳遁：巽→离→坤→兑→乾→坎→艮→震（宫位 [4,9,2,7,6,1,8,3]）
 - 阴遁：巽→震→艮→坎→乾→兑→坤→离（宫位 [4,3,8,1,6,7,2,9]）
- 按照外围八宫顺序，先记住每个宫对应的地盘干，在后台依次打印出地盘干
- 根据之前获得的"时柱"，在下面的对应表中查出"旬首"
| 旬首 |          时柱                                         |
| 戊   | 甲子、乙丑、丙寅、丁卯、戊辰、己巳、庚午、辛未、壬申、癸酉 |
| 己   | 甲戌、乙亥、丙子、丁丑、戊寅、己卯、庚辰、辛巳、壬午、癸未 |
| 庚   | 甲申、乙酉、丙戌、丁亥、戊子、己丑、庚寅、辛卯、壬辰、癸巳 |
| 辛   | 甲午、乙未、丙申、丁酉、戊戌、己亥、庚子、辛丑、壬寅、癸卯 |
| 壬   | 甲辰、乙巳、丙午、丁未、戊申、己酉、庚戌、辛亥、壬子、癸丑 |
| 癸   | 甲寅、乙卯、丙辰、丁巳、戊午、己未、庚申、辛酉、壬戌、癸亥 |
表把时柱直接映射到旬首（戊/己/庚/辛/壬/癸）
- 通过时柱查旬首，把旬首落到与时干对应地盘干的宫，这个宫位上作为天盘干的起始点；
- 如果遇到时干是"甲"的情况，旬首对应地盘干的宫作为天盘干的起始点；
- 按照刚才按照外围八宫顺序记住的地盘干，以旬首找到所在地盘干的位置，从这个位置开始把地盘干依次填满剩余天盘干；


##天盘十干是不允许真正停在中五宫
- 在奇门遁甲里，天盘十干是不允许真正停在中五宫的。
- 当你排天盘干时：如果计算出来的“起点天干”（旬首、时干所对应的天盘干）落在中 5 宫，一律采用「出中入坤」的规则——寄居坤二宫，即若结果是中五宫，就不要把天干放在中宫，而是把这个起点天干移到坤二宫
- 然后再从这个宫位（坤二宫）出发，按阳遁顺行、阴遁逆行的方向，继续把其余天干依次排满各宫。



##核心功能
- 创建后台程序 app/api/tianpangan/route.ts 
- 前台页面： app/page.tsx  
- 前台页面中，点击"排盘"后，在排地盘干后，执行排天盘干
- 排盘顺序：定局 → 地盘干 → 天盘干
